The fastest way to edit Markdown & MDX on the web

A from-scratch, zero-dependency editor engine with an Obsidian-style unified live preview, a full formatting toolbar, inline comments, semantic folding, and an LLM stream renderer that anticipates formatting as it arrives. Drop-in React component.

Try the demo ↓ $ npm install typewright

Render an LLM stream while anticipating formatting

Feed Typewright a token stream and it renders word-by-word — optimistically resolving incomplete formatting so the preview reads as finished prose while it's still arriving. Confirmed text never reflows.
